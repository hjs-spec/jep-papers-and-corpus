# Version Relationship

This repository contains supporting research and corpus material for the JEP / HJS / JAC protocol stack.

It is not the normative source for current protocol behavior.

## Current Version Line

| Layer | Current Version | Role |
|---|---|---|
| JEP | v0.6 | Stable event core: Core / Profiles / Conformance |
| JEP API | v0.6 | API implementation seed for JEP-Core events |
| HJS | v0.5 | Accountability receipt and evidence-lifecycle companion layer |
| JAC | v0.5 | Declared dependency chain companion layer |

## Repository Type

```text
research-supporting-material
```

This repository may include historical language, exploratory corpora, and research papers that predate the current version line.

Where this repository conflicts with current drafts or implementation repositories, the current drafts and repositories take precedence.
